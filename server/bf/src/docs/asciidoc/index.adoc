== board 페이지
=== 게시글 등록
카테고리가 EATING이면 genderTag와 foodTag를 모두 등록할 수 있다.
만약 genderTag를 등록하지 않으면 기본값인 3(남녀노소)로 등록되고 foodTag를 등록하지 않으면 기본값인
5(기타)로 등록된다.

카테고리가 SHOPPING이면 genderTag만 등록할 수 있다.
이때도 위와 마찬가지로 genderTag를 등록하지 않으면 기본값인 3(남녀노소)로 등록된다.

=== 게시글 검색 (검색어, 태그)
- 한페이지당 게시글 10개씩 최신순
- 제목이나 내용에 검색어가 포함되어있으면 조회
- category가 값이 없으면 기본으로 EATING
- 성별 태그로 검색하고 싶으면 genderTag (1,2,3)
- 음식 태그로 검색하고 싶으면 foodTag (1,2,3,4,5)
